🚀 Deploy e Messa Online su Hostinger VPS

Per ospitare il Politometro con salvataggio dati attivo sul tuo server VPS KVM 1 di Hostinger, segui questi passi:

1. Variabili d'Ambiente:
Configura le variabili d'ambiente nel file .env o direttamente nel terminale per blindare l'accesso admin:
POLITOMETRO_ADMIN_EMAIL=tua_mail@dominio.com
POLITOMETRO_ADMIN_PASSWORD_HASH=hash_della_tua_password (crealo usando lo script apposito in backend).
2. Lancio del Server FastAPI backend:
Avvia il backend di produzione tramite gunicorn/uvicorn sulla porta 8000:
.venv-politometro/bin/uvicorn backend.main:app --host 127.0.0.1 --port 8000
3. Configurazione Nginx/Caddy proxy:
Imposta il web server in modo che le richieste sotto /api e /admin vengano inoltrate al backend FastAPI (porta 8000), mentre le richieste statiche servano i file presenti in dist/.
